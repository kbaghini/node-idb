# Code of conduct

## Our standard

We are committed to a respectful, inclusive, and technically constructive
community. Examples of positive participation include:

- Showing empathy and patience
- Giving actionable, evidence-based feedback
- Respecting differing experience levels and viewpoints
- Focusing disagreement on ideas and implementation details
- Accepting responsibility and correcting mistakes

Unacceptable behavior includes harassment, discrimination, threats, personal
attacks, sexualized content, deliberate disruption, or publishing another
person's private information without permission.

## Enforcement

Repository maintainers may edit or remove contributions that violate this
standard and may temporarily or permanently restrict participation. Enforcement
decisions should be proportional, documented privately, and focused on keeping
the project safe and productive.

Report conduct concerns privately through the repository owner's GitHub
profile. Do not use public issues to disclose sensitive personal information.

This code applies in repository spaces and when representing the project in
public spaces.
